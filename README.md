# Jesus Generation Mini App

## Executive Summary
Jesus Generation Mini App is the member-facing digital experience for Jesus Generation Community Ministry.

The project is designed to support spiritual growth, onboarding, communication, discipleship pathways, and leadership development through a clean mobile-first experience.

## Strategic Purpose
This app is part of a larger digital ministry ecosystem built to help people move from connection to discipleship to leadership.

It is intended to work alongside:

- JGC Pastor Dashboard
- Shepherd's Bridge
- 242go
- Leadership Pipeline systems

## Core Outcomes
- Help new people connect quickly
- Guide members through next steps
- Support onboarding and discipleship
- Centralize ministry communication
- Create pathways for serving and leadership growth
- Reduce administrative load while preserving human care

## Planned Features
- Member profile
- New member onboarding
- Ministry next steps
- Prayer request flow
- Serving interest form
- Leadership growth pathway
- AI-assisted guidance within approved ministry guardrails
- Dashboard integration for leaders

## Technology Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
- Telegram Mini App SDK
- Notion integration
- Anthropic SDK

## Product Positioning
This is not just a church app. It is a ministry operating layer designed to help churches care for people, develop leaders, and organize discipleship with clarity.
